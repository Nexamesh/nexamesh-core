## Application Notes by Program

### NVIDIA Inception
- URL: nvidia.com/inception
- Application: Simple form. Company name, product description, use of NVIDIA hardware.
- Key claim: "Edge AI drone detection on NVIDIA Jetson using YOLOv8. Nexamesh detector runs on Jetson Nano at R1,500 hardware cost for African markets."
- Approval: Usually within 48 hours.
- What you get: Hardware discounts, DGX Cloud credits, co-marketing, VC network access.

### AWS Activate (Founders Tier)
- URL: aws.amazon.com/activate/founders
- Application: Company name, stage (pre-seed), use case.
- Key claim: "AI inference API + blockchain evidence anchoring on AWS. Nexamesh runs Rust (Axum) API and Solana/EtherLink evidence anchoring services."
- Approval: Usually instant or within 1 week.
- What you get: $1K–$5K credits (Founders tier). Portfolio tier ($25K) requires AWS investor relationship.

### Microsoft Founders Hub
- URL: foundershub.startups.microsoft.com
- Application: Standard form. Company info, founder details, product stage.
- Key claim: "Nexamesh is building AI-powered drone detection for African critical infrastructure. Edge inference + blockchain evidence anchoring. Currently pre-seed."
- Approval: 1–2 weeks.
- What you get: Up to $150K Azure credits over 2 years, plus LinkedIn credits, GitHub Enterprise, OpenAI credits.
- **CRITICAL**: Charl applies, not Jurie. New brand, new email, fresh application.

### Google for Startups Africa
- URL: startup.google.com/programs/black-founders-fund/africa (or main Africa program)
- Application: Cohort-based. Requires pitch video or written application. More competitive than above.
- Key claim: Lead with AI (YOLOv8 detector) and African market (critical infrastructure in SA, SADC).
- What you get: $200K Google Cloud credits, mentorship, network access.

### TIA Seed Fund
- URL: tia.org.za → Funding → Seed Fund
- Application: Formal document submission. Tech overview, team CVs, market analysis, budget.
- **Prerequisite**: CSD registration + SA Pty Ltd + business bank account.
- Key claim: "Pre-commercial counter-UAS technology with blockchain evidence anchoring. Novel technology, SA-developed, defense/security sector."
- Non-dilutive: TIA takes royalty on future revenue. No equity given up.
